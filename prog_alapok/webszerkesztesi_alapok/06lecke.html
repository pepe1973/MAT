Hatodik lecke - Visual Studio Code beállítása

  1. Honnan tudjuk letölteni a Visual Studio Code-ot?

    • Látogassunk el a hivatalos oldalára és onnan töltsük le a megfelelő böngészőhöz az alkalmazást. Mi csak a Windows-os telepítést nézzük meg.
    • Forrás: Visual Studio Code
  2. Hogyan telepítsük a Visual Studio Code-ot?

    • Fogadjuk el a szerződési feltételeket.
    • Jelöljünk be minden opciót.
    • Indítsuk el a telepítést.
  3. Milyen kezdeti beállítások javasoltak?

    • A lenti példában az látható, miszerint elérhető az, hogy a Visual Studio Code-on belüli Explorer-ben a beágyazott mappák nevei egymás alatt és nem egymás mellett szerepeljenek, amely elég hasznos tud lenni.
    • Ehhez nyissuk meg a bel alsó sarokban lévő fogaskerékre kattintva a Setings-fület.
    • Gépeljük be a keresőbe a következőt: explorer.compact folders. Majd vegyük ki a pipát a jelzett helyen.
  4. Mely Extension-ők segíthetnek a webszerkesztésben?

    • Számos kényelmi funkciót, ugynevezett Extension-t szolgáltat a Visual Studio Code. Ehhez kattintsuk az ikonjára a baloldalon. Majd írjuk be az keresett Extension nevét. És telepítsük.
    • Néhány az ismertebbek közül. Olvassuk el, hogy mit tud az adott Extension.
    • És íme még néhány.
  5. Hogyan lehet beállítani a Prettier-t?

    • A Prettier azért felelős, hogy ha helyes a kód, akkor megfelelő módon fogja eltörni, hogy olvashatóbb legyen. Ehhez ismét nyissuk meg a Settings-et.
    • Majd gépeljük be a formatter szót. Ezután a legördülő listában keressük meg a Prettier-t.
    • Már csak az alatta lévő négyzeteket kell bejelőlni.
  6. Mi az az Emmet?

    • Az Emmet olyan beépülő modulok készlete szövegszerkesztőkhöz, amelyek nagy sebességű kódolást és szerkesztést tesznek lehetővé HTML, XML, XSLT és más strukturált kódformátumokban a tartalomsegítő segítségével.
    • Forrás: Emmet
    • Forrás: Emmet a Visual Studio Code-ban
    • Forrás: Emmet Cheat Sheet
  7. Mi az a HTML CSS Support?

    • Egy Extension a Visual Studio Code-ban.
    • Ha a html állományban már létező css osztályokra akarok hivatkozni, akkor kód kiegészítéssel ezt megoldhatom.
    • Ehhez a külső vagy belső stíluslapon létrehozom az osztályt, majd az adott elemben a class="" attribútumban elkezdem begépelni az osztálykijelölő nevét, amit aztán a Tab lenyomásával kiegészíthetek.
    • Ugyanez igaz elemkijelölők (id) esetén is.
